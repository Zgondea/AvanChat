🏛️ Chat Legislativ - Demo Integrat

Bun venit la demonstrația widget-ului Chat Legislativ! Acest sistem oferă asistență AI pentru întrebări despre legislația fiscală locală pentru primăriile din România.

👀 Vezi chat-ul în acțiune! Butonul de chat este deja activ în colțul din dreapta jos al acestei pagini. Faceți click pe el pentru a testa funcționalitatea.

🚀 Integrare Rapidă

Pentru a integra widget-ul pe site-ul primăriei, adăugați următoarele linii de cod:

<!-- Adăugați în <head> --> <link rel="stylesheet" href="https://your-domain.com/widget/fiscal-chat-widget.css"> <!-- Adăugați înainte de </body> --> <script src="https://your-domain.com/widget/fiscal-chat-widget.js" data-chat-legislativ data-municipality-domain="primaria-bucuresti.ro" data-api-url="https://your-api-domain.com/api/v1" data-title="Asistent Fiscal Primăria București" data-welcome-message="Bună ziua! Vă pot ajuta cu întrebări despre taxele și impozitele locale." ></script>

⚙️ Opțiuni de Configurare

Parametru Descriere Valoare Implicită
data-api-url URL-ul API-ului backend http://localhost:8000/api/v1
data-municipality-domain Domeniul primăriei pentru identificare automată window.location.hostname
data-municipality-id ID-ul specific al primăriei (alternativă la domain) null
data-primary-color Culoarea principală a widget-ului #1976d2
data-title Titlul afișat în header-ul chat-ului Asistent Legislativ
data-welcome-message Mesajul de bun venit Mesaj standard despre taxe și impozite

🎯 Funcționalități

  • RAG (Retrieval-Augmented Generation) - Răspunsuri bazate pe documente oficiale
  • Citarea surselor - Fiecare răspuns include referințe la documentele sursă
  • Contextul conversației - Asistentul își amintește conversația curentă
  • Responsive design - Funcționează pe desktop și mobile
  • Personalizare - Culori și mesaje personalizabile pentru fiecare primărie
  • Multi-tenant - Suport pentru mai multe primării cu documente diferite

🔧 Configurare Avansată (JavaScript)

Pentru configurare programatică:

// Inițializare manuală const chatWidget = window.ChatLegislativWidget.init({ apiUrl: 'https://your-api-domain.com/api/v1', municipalityId: 'uuid-here', primaryColor: '#2196f3', title: 'Asistent Fiscal Personalizat', welcomeMessage: 'Mesaj personalizat de bun venit!', enableSources: true, enableTyping: true }); // Accesare instanță existentă const widget = window.ChatLegislativWidget.getInstance(); // Distrugere widget window.ChatLegislativWidget.destroy();

🛠️ Arhitectura Sistemului

  • Backend: FastAPI + PostgreSQL + pgvector
  • AI Model: Ollama cu Llama 3.2 3B (local)
  • Embeddings: sentence-transformers (multilingv pentru română)
  • Frontend Admin: React + Material-UI
  • Widget: Vanilla JavaScript (fără dependențe)
  • Containerizare: Docker + Docker Compose
